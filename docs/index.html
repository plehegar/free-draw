This is the generated documentation for Free Draw.

Move functions

Those functions will move the pen while drawing.

moveBackward(distance)

move backward given a certain distance

moveBackward(50);

moveForward(distance)

move forward given a certain distance

moveForward(100);

moveTo(x, y)

move forward to a certain point

moveTo(10, 10);

moveToCenter()

back to the center, ready to move up

moveToCenter();

Jump functions

Those functions will move the pen without drawing.

jumpBackward(distance)

jump backward given a certain distance

jumpBackward(50);

jumpForward(distance)

jump forward given a certain distance

jumpForward(50);

jumpTo(x, y)

jump to a certain point

jumpTo(10, 10);

Pen functions

Those functions set the properties of the pen (color, etc.).

penCap(cap)

set the cap of the pen. cap is one of "butt", "round", "square"

penCap("square");

penColor(color)

change the color of the pencil (use "random" for a random color)

penColor("blue");

penWidth(width)

set the width of the pen

penWidth(10);

Draw functions

Those functions draw shapes and images.

rect(width, height, [fill])

draw a rectangle

rect(20, 10, "red");

show(character, [effect])

display a character with an optional effect ("mirror", "big", etc.), above the current drawing.

show("knight"); show("princess", "small");

text(text, [fontStyle])

write a text in the drawing.

text("Hello"); text("Hello", "20px sans-serif");

clearPaint()

clear the drawing and move back to center

clearPaint();

Direction functions

Angles use degrees (°): One full turn is equal to 2π radians (2π rad) and is equal to 360 degrees (360°).

turnLeft(degree)

turn counter-clockwise given a certain number of degree

turnLeft(45);

turnRight(degree)

turn clockwise given a certain number of degree

turnRight(90);

direction(degree)

set the direction to a certain number of degree

direction(90);

Events functions

when(eventName, [keyboard], function)

react to events "mouse", "keyboard", and "tick"

function f(x, y) { ... };
when("mouse", f);
function f(key) { ... };
when("key", "a", f);
when("key", "all", f);
function fct() { ... };
when("tick", fct);

clearEvents()

Clear all events.

clearEvents();

random(1, 10);

randomColor();
